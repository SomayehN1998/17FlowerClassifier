# 17FlowerClassifier

این پروژه برای شناسایی نوع گل از روی تصاویر استفاده از شبکه عصبی کانولوشنی (CNN) است. دیتاست **17Flower** که شامل تصاویر 17 نوع مختلف گل می‌باشد، برای آموزش مدل استفاده می‌شود.

## توضیح پروژه

هدف این پروژه، آموزش یک مدل CNN برای شناسایی 17 نوع گل مختلف است. مدل با استفاده از دیتاستی که شامل تصاویر گل‌ها در دو بخش **train** و **test** است آموزش داده می‌شود. این مدل می‌تواند برای شناسایی انواع مختلف گل‌ها به کار رود.

## ویژگی‌ها

- استفاده از **کتابخانه Keras** برای پیاده‌سازی مدل.
- دقت بالای مدل تا 90٪ برای شناسایی گل‌ها.
- امکان شبیه‌سازی و نمایش تصاویر خروجی گل‌ها.
- استفاده از دیتاست **17Flower** برای آموزش مدل.
